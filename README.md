#fx-payroll

> 放薪管家2.0版--微信工资条

## Build Setup

``` bash
# 安装依赖
npm install

# 端口号:8017
npm run dev

# 打包sit环境
npm run sit

# 打包demo环境
npm run demo

# 打包正式环境
npm run build

```
